# re:SearchTX Support Playbook

This playbook provides a comprehensive reference for troubleshooting, understanding, and supporting visibility, indexing, and security behavior in the re:SearchTX platform. The content is based on the JCIT Technology Standards v10.0 and internal system behavior within re:SearchTX, the Document Access System (DAS), and CMS/EFM integrations.

This documentation is intended for BIS staff, support analysts, CMS vendors, EFSPs, partner teams, and anyone responsible for diagnosing or explaining data behavior observed in re:SearchTX.
