# Client Documentation

This section contains all high-level, business-focused materials for courts, CMS vendors, and integrators working with the **re:Search** platform.

It includes:

- Overviews of all re:Search integration modes  
- Guidance for choosing the correct integration model  
- Market-level documentation for Batch, ECF, CIP (EJ), and non-integrated courts  
- End-to-end onboarding guidance for new CMS partners and courts  

Use this folder to understand **how re:Search connects to different CMS systems**, what integration models are supported, and how onboarding works from a business/functional perspective.
