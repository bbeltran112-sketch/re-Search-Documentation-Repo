# Client Documentation

**Navigation:**  
[Home](../../README.md) › Client Documentation

This section provides all guidance for **courts**, **CMS vendors**, and **EFSP integrators** who work with re:SearchTX.  
It explains integration models, onboarding steps, certification expectations, and decision-making resources.

---

## Purpose

Client Documentation provides:

- Market-level context for re:Search deployments  
- Detailed descriptions of each integration mode  
- Selection guidance for CMS vendors  
- Court onboarding workflows  
- Terminology and expected data behaviors  

These pages help new partners understand the end-to-end integration lifecycle.
