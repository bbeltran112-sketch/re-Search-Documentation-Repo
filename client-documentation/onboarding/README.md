# Client Onboarding – re:Search Platform

This folder contains all onboarding materials used to guide CMS vendors, courts, and Tyler internal teams through the full lifecycle of integrating with **re:Search**.

The onboarding flow is designed to lead clients from initial preparation → environment setup → integration selection → API implementation → testing → go-live.
