## Overview

This page lists all downloadable resources, templates, and tools available during the onboarding process. Most resources are provided by your Technical Project Manager at the appropriate phase.

---

## Planning & Preparation Resources

### CMS Preparation Checklist
**Format:** Excel (.xlsx)  
**When Available:** Phase 1  
**Purpose:** Assess your CMS technical readiness for integration

**Includes:**
- API capability assessment
- Data export capabilities
- Security handling verification
- Performance requirements
- Network/firewall checklist

**How to obtain:** Request from your TPM during Phase 1 kickoff

---

### Integration Mode Decision Matrix
**Format:** PDF  
**When Available:** Phase 2  
**Purpose:** Visual comparison of all integration modes

**Includes:**
- Mode comparison table
- Decision tree flowchart
- Capability requirements by mode
- Timeline estimates

**How to obtain:** Request from your TPM or download from [Integration Modes Overview](../integration-modes/README.md)

---

## Development Resources

### Technical Onboarding Slide Deck
**Format:** PowerPoint (.pptx)  
**When Available:** Phase 3  
**Purpose:** Developer training and API overview

**Includes:**
- Architecture overview
- API specifications by mode
- Authentication setup
- Error handling patterns
- Code examples

**How to obtain:** Provided during Phase 3 technical kickoff meeting

---

### API Postman Collection
**Format:** JSON (Postman)  
**When Available:** Phase 3  
**Purpose:** Pre-built API requests for testing

**Includes:**
- All API endpoints with examples
- Environment variables template
- Authentication configuration
- Sample request/response pairs

**How to obtain:** Request from your TPM after receiving test credentials

---

### Sample Test Data
**Format:** JSON / XML  
**When Available:** Phase 3  
**Purpose:** Test cases for development and validation

**Includes:**
- Sample case records (public, confidential, sealed)
- Filing examples
- Party/attorney data
- Document metadata

**How to obtain:** Provided with test environment access

## How to Request Resources

**During onboarding:**
1. Contact your assigned Technical Project Manager
2. Specify which resource you need
3. Indicate which phase you're currently in

**After go-live:**
- Email BIS Team at [bryce.beltran@tylertech.com](mailto:bryce.beltran@tylertech.com)
- Submit request through Tyler Partner Portal
